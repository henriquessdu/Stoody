🎮 ESTADO GLOBAL (GameContext)
═══════════════════════════════════════════════════════════════════════════

Estados:
  • xp                 Pontos de experiência
  • level              Nível atual (1-100+)
  • xpMax              XP necessário para próximo nível
  • coins              Moedas coletadas
  • search             Busca de cursos
  • completedCourses   Array de IDs de cursos completos
  • levelPulse         Flag para animação de level-up
  • coinAnim          Flag para animação de moedas

Funções:
  • addXP(amount)               Adiciona XP (sobe nível automaticamente)
  • addCoins(amount)            Adiciona moedas
  • completeCourse(courseId)    Marca curso como completo
  • isCourseCompleted(courseId) Verifica se já foi feito
  • resetProgress()             Reseta tudo (logout)

📊 SISTEMA DE NÍVEL
═══════════════════════════════════════════════════════════════════════════

Level 1:  0 XP   → 100 XP  (para Level 2)
Level 2:  0 XP   → 130 XP  (para Level 3)
Level 3:  0 XP   → 169 XP  (para Level 4)
Level 4:  0 XP   → 220 XP  (para Level 5)
...

Fórmula: xpMax *= 1.3 a cada nível

Exemplo Real:
  • Completa course 1: +150 XP → Level 2, XP restante: 50
  • Completa course 2: +120 XP → Total 170, sobe para Level 3
  • Navega, volta → dados persistem em localStorage

🔄 FLUXO COMPLETO
═══════════════════════════════════════════════════════════════════════════

1. Usuário abre app
   └─ Login.jsx (valida email)

2. Clica "Começar Agora"
   └─ Home.jsx (mostra 3 cursos)

3. Escolhe um curso (ex: Japanese for Anime)
   └─ CoursePlayer.jsx ESTADO 1 (vídeo)
   └─ Assiste vídeo

4. Clica "Começar Prova"
   └─ CoursePlayer.jsx ESTADO 2 (quiz)
   └─ Responde 5 perguntas
   └─ QuestionCard recebe feedback visual

5. Termina quiz
   └─ CoursePlayer.jsx ESTADO 3 (completion)
   └─ CompletionCard mostra recompensas
   └─ XP é adicionado automaticamente
   └─ Coins são adicionadas automaticamente
   └─ Navbar atualiza em tempo real

6. Clica "Finalizar"
   └─ Volta para Home.jsx
   └─ Curso aparece como "Já Completado"

7. Próxima tentativa no mesmo curso
   └─ CoursePlayer.jsx ESTADO "COMPLETO"
   └─ Apenas vídeo é mostrado
   └─ Mensagem: "Você já completou esta lição"
